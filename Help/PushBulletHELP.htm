PUSHBULLET HELP

 

 

BOT HELP - send this help message

BOT DELETE  - delete all your previous Push message

BOT START - start the bot named

BOT <Village Name> RESTART - restart the bot named <Village Name> and emulator

BOT <Village Name> STOP - stop the bot named <Village Name>

BOT <Village Name> PAUSE - pause the bot named <Village Name>

BOT <Village Name> RESUME   - resume the bot named <Village Name>

BOT <Village Name> STATS - send Village Statistics of <Village Name>

BOT <Village Name> LOG - send the current log file of <Village Name>

BOT <Village Name> LASTRAID - send the last raid loot screenshot of <Village Name>

BOT <Village Name> LASTRAIDTXT - send the last raid loot values of <Village Name>

BOT <Village Name> SCREENSHOT - send a screenshot of <Village Name>

BOT <Village Name> SENDCHAT <TEXT> - send TEXT in clan chat of <Village Name>

BOT <Village Name> GETCHATS <STOP|NOW|INTERVAL> - select any of this three option to do

BOT <Village Name> RESETSTATS - reset Village Statistics of <Village Name>

BOT <Village Name> HALTATTACKON - Turn On 'Halt Attack' in the 'Misc' Tab with the default options

BOT <Village Name> HALTATTACKOFF - Turn Off 'Halt Attack' in the 'Misc' Tab

BOT <Village Name> SWITCHPROFILE <PROFILENAME> - Swap Profile Village and restart bot

BOT <Village Name> GETCHATS <INTERVAL|NOW|STOP> - to get the latest clan chat as an image

BOT <Village Name> SENDCHAT <chat message> - to send a chat to your clan

BOT <Village Name> RESETSTATS - reset Village Statistics

BOT <Village Name> HALTATTACKON - Turn On 'Halt Attack' in the 'Misc' Tab with the default options

BOT <Village Name> HALTATTACKOFF - Turn Off 'Halt Attack' in the 'Misc' Tab

 

BOT <Village Name> SCREENSHOTHD - send a screenshot in high resolution of <Village Name>

BOT <Village Name> BUILDER - send a screenshot of builder status of <Village Name>

BOT <Village Name> SHIELD - send a screenshot of shield status of <Village Name>

BOT <Village Name> TOP - send a top gain statistics of <Village Name>

 

BOT ACC <acc list> - set new play list

BOT ADD <acc number> - add an account to play list

BOT REM <acc number> - remove an account from play list

BOT MAP <acc number>-<pro number> - set profile to an account. eg: BOT MAP 1-3

BOT MODE <mode ID> - set switching mode. Eg: BOT MODE 0

BOT PRO <pro number> - set new bot profiles

BOT GETORDER - get current CoC account and bot profile

BOT STOPSTART - stop then start bot again

BOT ALLPRO <all pro number> - set up profiles correspond to all exists accounts

BOT HIDE - hide android emulator

BOT ATKP 1/0 - 1-enable/0-disable attack plan
